Benjamin's Navigation 现已支持深色模式

2021-08-20 3 min read # 更新

在昏暗的环境中，打开深色模式可获得出色的视觉体验

Benjamin 已于今日推送了 Benjamin's Navigation 的全新版本，为其带来了对深色模式（Dark Mode）的支持以及更多改进与修复

深色模式

「深色模式」带来了全然一新的外观，它能使您的眼睛放松，并有助于您专心浏览。深色模式使用一种较深的配色方案，这种配色作用于整个页面，包括汉堡导航栏和导航项目内容

在深色模式下浏览 Benjamin's Navigation

此次对深色模式的支持不仅体现在页面总体背景色，所有导航项目和地址链接 Pop 均已支持深色模式

现已为各个导航项目实现了深色模式下的正确显示

深色模式的优势

  • 可大幅减少耗电量（具体取决于设备的屏幕技术，如 OLED 屏幕）；
  • 为弱视以及对强光敏感的用户提高可视性；
  • 让所有人都可以在光线较暗的环境中更轻松地使用设备

开启深色模式

想要在全新版本的 Benjamin's Navigation 中使用深色模式进行浏览，需要您为您的浏览设备开启系统级深色模式并使用支持媒体查询的网页浏览器进行访问，因为在实现方式上，使用了 prefers-color-scheme 这一 CSS 媒体查询技术

其它改进与修复

除了加入对深色模式的支持，本次更新还改进了汉堡导航栏在站点文章页面的显示，提供了一键返回首页的功能

在文章页面，汉堡导航菜单已提供一键返回首页功能

另外，作为周期性更新维护项目，本次更新还修改了部分内容的分类并修复了一系列小问题
