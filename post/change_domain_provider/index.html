Benjamin 已更改所属域名的解析服务提供商

2022-08-02 1 min read # 新闻稿

所属域名的解析服务现已由阿里云变更为 Cloudflare

Benjamin 已于近日更改了其所属域名的解析服务提供商。现在，当你访问 Benjamin 的所属域名及相关网站时，Cloudflare 将为你域名解析服务

由于原先的域名解析服务提供商阿里云存在透明度不明等问题，出于后期维护需要，Benjamin 已将其所属域名解析服务提供商有阿里云更改为 Cloudflare

由于实际需要，Cloudflare 仅为 Benjamin 的所属域名提供 DNS 服务，并不提供其它服务选项。由于 Cloudflare 的 DNS 服务在中国大陆地区可能存在访问问题，使用来自中国大陆地区的 IP 地址访问 Benjamin 所属域名及相关网站时可能存在稳定性问题。由此产生的问题，Benjamin 和 Cloudflare 不对此负责，亦不再对相关反馈作出回应
