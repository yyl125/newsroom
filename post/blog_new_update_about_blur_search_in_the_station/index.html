Benjamin's Blog 现已支持站内模糊搜索

2021-05-22 2 min read # 更新

搜素结果暂时仅供参考

Benjamin 已于近日推送了全新版本的 Benjamin's Blog，为其带来了站内模糊搜索功能

站内模糊搜索

通过站内模糊搜索，访客可以更加快速的搜索到 Benjamin's Blog 的内容

现已上线站内模糊搜索

当前状况

由于引用了搜索 API 这一开源内容，受技术条件的限制，在执行第一次搜索时可能需要耗费较长时间构建索引，后续会直接从索引中提取相关内容。搜索结果目前仅能作为参考，搜索出的内容与实际提交的关键词可能并无明显关系

搜索到的关键词会在搜索结果中以高亮显示

搜索功能的范围仅包括文章，而搜索到的关键词会在搜索结果中以高亮显示

后续计划

后续计划继续对搜索 API 进行优化，争取完善站内模糊搜索功能

其它改进和修复

另外，作为周期性更新维护项目，本次更新还修复了一系列小问题
